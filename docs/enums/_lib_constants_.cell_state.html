Enumeration members

DEFAULT

DEFAULT:

Initial state of all Cells. Defined as the absence of all other state.

FLAGGED

FLAGGED:

The user has placed a flag on this MinesweeperCell. Flagged Cells are not revealed when the user clicks on them.

QUESTIONED

QUESTIONED:

The user has marked this MinesweeperCell with a question mark indicating that they are unsure whether or not this MinesweeperCell contains a mine.

REVEALED

REVEALED:

The user has revealed this MinesweeperCell. Once a MinesweeperCell is revealed, the user can no longer interact with a cell.
